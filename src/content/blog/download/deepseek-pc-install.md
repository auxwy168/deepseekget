## Windows 电脑版安装步骤

### 第一步：下载安装包

打开 DeepSeek 官网或本站下载页面，选择 Windows 版本下载。文件名格式为 `DeepSeek-Setup-x.x.x.exe`，大小约 120MB。建议优先从官网下载以确保安全——正版渠道可以参考本站的[DeepSeek官网正版入口辨别指南](/download/deepseek-official-site/)。

### 第二步：运行安装程序

双击下载好的 .exe 文件。Windows 可能弹出"Windows 已保护你的电脑"提示，点击"更多信息"→"仍要运行"即可。这是正常的安全提示，因为 DeepSeek 客户端尚未加入微软数字签名白名单。

### 第三步：选择安装路径

默认安装在 `C:\Users\用户名\AppData\Local\DeepSeek`。想要安装到其他盘，点击"自定义安装"选择目录。安装过程约 1-2 分钟。

### 第四步：登录并开始使用

安装完成后桌面会自动创建快捷方式。双击打开，使用手机号或邮箱注册登录。登录后对话记录会在手机端和电脑端自动同步。手机端安装可以查看[DeepSeek手机APP下载注册指南](/tutorial/deepseek-mobile-guide/)。

## Mac 电脑版安装步骤

### 第一步：下载 dmg 安装包

在下载页面选择 macOS 版本，下载 dmg 文件约 140MB。支持 macOS 12（Monterey）及以上版本。

### 第二步：拖入应用程序文件夹

双击 .dmg 文件，将 DeepSeek 图标拖入 Applications 文件夹。首次打开时如果提示"无法验证开发者"，前往「系统设置 → 隐私与安全性」→ 点击「仍要打开」。

### 第三步：登录并同步

打开后用手机号或邮箱登录。如果你在手机上已经在使用 DeepSeek，登录后对话历史会自动同步到 Mac 客户端。

## 常见问题

### 电脑版和网页版有什么区别？

电脑版启动更快、支持系统快捷键、可离线查看历史对话、支持消息通知。网页版无需安装，打开即用。两者的模型能力完全相同。

### 安装后打不开怎么办？

检查 Windows 版本是否为 Windows 10 1903 或更高版本。如果双击无反应，尝试右键→以管理员身份运行。Mac 用户需要在系统设置中允许来自未知开发者的应用。

### 电脑版需要付费吗？
