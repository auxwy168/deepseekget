第一步：注册与获取API Key

访问 DeepSeek 开放平台（platform.deepseek.com），使用邮箱或手机号注册。进入控制台「API Keys」页面创建新密钥。为密钥添加备注（如"测试环境"），便于后续管理。

⚠️ 重要：完整密钥仅在创建时显示一次，请立即保存到密钥管理系统。切勿将密钥写入公开代码仓库——用环境变量或密钥管理工具。

第二步：基础调用

DeepSeek API兼容Chat Completions风格接口。请求头携带 Authorization: Bearer YOUR_API_KEY，正文指定模型名称和消息数组。首次集成先用短prompt验证连通性，再逐步放开max_tokens做长文测试。

第三步：模型选择

  • deepseek-chat（V3）：通用对话，¥1/百万token，性价比最高
  • deepseek-reasoner（R1）：深度推理，¥4/百万token，适合复杂分析
  • Coder系列：代码补全和审查，适合IDE插件集成

对比ChatGPT GPT-4o约¥36/百万token，DeepSeek便宜约15-36倍。新用户注册赠送免费额度。

第四步：错误处理

  • 401：密钥无效或已过期，检查API Key
  • 429：速率限制，实现指数退避重试
  • 500：服务端异常，稍后重试
